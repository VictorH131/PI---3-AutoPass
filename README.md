# **Projeto Integrador – Fatec Itapira (2026) -  AutoPass**     

### Sistema Inteligente de Estacionamento e Acesso Automatizado com IoT

Projeto Integrador desenvolvido para a Faculdade de Tecnologia do Estado de São Paulo - Fatec Itapira, com foco em automação comercial, mobilidade inteligente e Internet das Coisas (IoT).

---

## 📖 Sobre o Projeto

O AutoPass é uma plataforma inteligente para gerenciamento de estacionamentos, desenvolvida para automatizar o controle de acesso de veículos, monitoramento de vagas e gerenciamento operacional através da integração entre sistema web e dispositivos IoT.

A solução busca reduzir filas, otimizar o fluxo de veículos e fornecer informações em tempo real para usuários e administradores.

---

## 🎯 Objetivo

Desenvolver um sistema capaz de:

* Automatizar entradas e saídas de veículos.
* Monitorar vagas em tempo real.
* Gerenciar usuários e veículos.
* Integrar sensores e dispositivos IoT.
* Simular pagamentos digitais automatizados.
* Melhorar a experiência dos usuários do estacionamento.
* Fornecer dados e relatórios para administradores.

## 🚨 Problema

Muitos estacionamentos ainda utilizam processos manuais ou sistemas pouco integrados, causando:

* Filas em horários de pico.
* Dificuldade para localizar vagas.
* Falhas em cobranças.
* Baixa visibilidade operacional.
* Falta de monitoramento em tempo real.

O AutoPass surge como uma alternativa moderna para resolver esses desafios através da automação e IoT.
